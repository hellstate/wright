# wright
Wright – A folder structure generator for web developers Description:
Wright is a simple platform designed for web designers and developers working with HTML, CSS, JavaScript, PHP, and related libraries and frameworks.

It automatically creates a ready-to-use folder structure, including: An ‍‍``index.html`` file for front-end development A ``dashboard/assets`` folder with subfolders for CSS, JS, images, and fonts A default stylesheet ``stylesheet.css`` and an empty app.
js Wright helps developers save time on naming and organizing folders, reducing confusion during project setup.

To run the file, make sure Node.js is installed, then use:

```bash node wright.js ```
